คู่มือระบบ PlanMarket
ระบบคืออะไร?
PlanMarket คือแอปพลิเคชันมือถือสำหรับ จองแผงตลาดนัด ที่เชื่อมต่อระหว่าง 4 กลุ่มผู้ใช้
Admin        — ดูแลระบบ อนุมัติผู้ใช้
Market Owner — เจ้าของตลาด จัดการแผง อนุมัติการจอง
Vendor       — ร้านค้า ค้นหาตลาด จองแผง
customer - ลูกค้าที่สามารถติดตามร้านค้าหรือตลาด

เทคโนโลยีที่ใช้
Frontend  │ Flutter (Dart)
Backend   │ Node.js + Express.js
Database  │ SQLite (planmarket.db)

สิ่งที่ต้องติดตั้ง
Backend
# 1. Node.js (แนะนำ v18+)
# ดาวน์โหลดที่ https://nodejs.org
# ตรวจสอบว่าติดตั้งแล้ว
node --version   # v18.x.x
npm --version    # 9.x.x

Frontend
# 1. Flutter SDK (แนะนำ v3.x+)
# ดาวน์โหลดที่ https://flutter.dev
# ตรวจสอบว่าติดตั้งแล้ว
flutter --version
# 2. ตรวจสอบ environment
flutter doctor

Tools เพิ่มเติม
Android Studio หรือ VS Code
Android Emulator หรือมือถือจริง
Git

วิธีติดตั้งโปรเจค
Step 1 — Clone โปรเจค
git clone <repository-url>
cd planmarket_all
Step 2 — ติดตั้ง Backend
cd backend
npm install
Step 3 — ติดตั้ง Frontend
cd ../frontend
flutter pub get

วิธีรันโปรเจค
 รัน Backend
# เปิด Terminal ที่ 1
cd backend
# ครั้งแรก — สร้างและ seed ข้อมูล
node db/reset-and-seed.js
node db/update-markets.js
# รัน server
npm run dev
# ควรเห็น
# Connected to SQLite database
# Database tables initialized
# Server running on port 3001

รัน Frontend
# เปิด Terminal ที่ 2
cd frontend

# ตรวจสอบ devices ที่เชื่อมต่อ
flutter devices

# รันแอป
flutter run

# หรือระบุ device
flutter run -d <device-id>

API Endpoints หลัก
Base URL: http://localhost:3001

Auth
POST   /auth/register        ← สมัครสมาชิก
POST   /auth/login           ← เข้าสู่ระบบ

Markets
GET    /markets              ← ดูตลาดทั้งหมด
GET    /markets/:id/stalls   ← ดูแผงในตลาด

Bookings
POST   /bookings             ← จองแผง
GET    /bookings/user/:id    ← ดูการจองของ vendor
PATCH  /bookings/:id/approve ← อนุมัติ
PATCH  /bookings/:id/reject  ← ปฏิเสธ

Market Favorites
GET    /market-favorites?userId=  ← ดูตลาดถูกใจ
POST   /market-favorites          ← เพิ่มถูกใจ
DELETE /market-favorites/:id      ← ลบถูกใจ
